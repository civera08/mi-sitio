Visualización - Creación de Dashboards

📊 Análisis Regional de Datos con Power BI

Visualización interactiva basada en datos procesados desde Excel.

🎯 Objetivo: Consolidar datos regionales desde múltiples archivos de Excel y presentarlos en un tablero visual que facilite la exploración por parte de usuarios no técnicos.

🛠️ Herramientas utilizadas:

  • Excel: limpieza y estructura base de datos
  • Power BI: modelo de datos, relaciones, segmentación interactiva y diseño visual

🎮 Reporte de Ventas por Consolas con Power BI

Visualización interactiva del desempeño de ventas segmentadas por tipo de consola, región y línea de productos.

🎯 Objetivo: Analizar y presentar de manera visual y dinámica las tendencias de ventas de consolas y accesorios a lo largo del tiempo, permitiendo identificar las plataformas más vendidas, comparar rendimiento mensual, y detectar oportunidades de crecimiento en distintas regiones mediante insights.

🛠️ Herramientas utilizadas:

  • Excel: depuración, limpieza y estructura base de datos
  • Power BI: modelo de datos, relaciones, segmentación interactiva y diseño visual, medidas DAX para KPI's

⭐ Aspectos destacados:

  • Filtros dinámicos por consola (PlayStation, Xbox, Nintendo, etc.)
  • Gráficos de líneas y columnas que muestran tendencias mensuales
  • KPIs que resumen ventas totales, rankings y comparaciones temporales
  • Navegación intuitiva para usuarios no técnicos

📎 Archivos del dashboard: Ver en GitHub
